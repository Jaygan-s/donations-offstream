# Donations Offstream
Twip 또는 Toonation 영상후원을 방송에 송출하지 않고 별도의 확장 프로그램으로 공유하는 시스템 (목표)

이것은 Go와 Javascript를 처음 쓰는 자의 피터지는 뚝배기-우선 개발에 관한 이야기
